shorthand-skill

Compress AI agent skill definitions by about 35-40% while preserving 100% of required data in evaluation.

What It Does

shorthand-skill provides a notation system for writing AI agent skill definitions with symbols, abbreviations, compact tables, and short section names.

Skills

SkillDescription
shorthandCore compression notation and rules.
shorthand-dictDictionary of symbols and abbreviations.
shorthand-commitCompressed Conventional Commit messages.
shorthand-compressConvert verbose skills to shorthand.
tool-setupApproval-gated dependency setup example.
port-scannerAuthorization-gated reconnaissance example.

Install

git clone https://github.com/khushalmistry/shorthand-skill.git
cd shorthand-skill
bash install.sh --agent codex

python3 evals/evaluate.py

Zero Evaluated Loss

Zero evaluated loss means every required fact in the preservation manifest appears in both the verbose baseline and shorthand skill. It does not mean identical wording; it means shorter notation carries the same required meaning.
